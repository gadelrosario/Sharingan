# Fantasy HQ Roadmap

## Product pillars
1. Consensus Intelligence
2. Sharingan Vision
3. Flight Control
4. Draft Windows
5. Opportunity Cost

## Draft identity
- Fantasy HQ: product
- Sharingan Vision: prediction engine
- Flight Control: concise guidance
- Chidori: rare must-draft-now alert

## In-season architecture
The same decision engine will power waivers, add/drop comparisons, trades, and start/sit decisions. Every move should return TeamFit, before/after roster strength, confidence, and a one-line explanation.

## Next integration stage
- Secure backend service
- Yahoo OAuth callback and encrypted token storage
- Yahoo league/team/roster/free-agent refresh
- Player ID mapping between Yahoo and Fantasy HQ
- Last-sync visibility, retries, and stale-data warnings
- Scheduled refresh around waivers and Sunday lineups
