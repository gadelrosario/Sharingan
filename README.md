# Gerard Fantasy HQ — Version 2.0 Alpha 12.3: League Personality Simulator

Install this folder directly; no earlier version is required. Upload all files and folders together to GitHub Pages or your existing host. Clear the old site cache once after deployment if the prior version remains visible.

## Alpha 12.3 improvements
- Preserves the Alpha 12.2 decision cards, working Sharingan Scan, filled-QB/TE logic, full player pool, roster tracker, wait meter, draft room scan, team exposure, and post-draft report.
- Gives every league manager a distinct baseline personality using the real profiles already provided.
- Randomizes a temporary strategy and draft-day mood for every opponent at the start of each mock.
- Adds controlled imperfection so lower-skill and less predictable managers occasionally reach, chase a run, choose a favorite player, or take their second-best option.
- Makes manager behavior react differently to roster needs, positional runs, rookies, upside, safe veterans, team affinity, stacks, and ADP.
- Keeps K and D/ST late while allowing occasional realistic early selections from less disciplined managers.
- Widens the score distribution by producing more varied, human-looking rosters instead of nine similarly optimized teams.

This is a bridge version until real-draft exports can provide stronger manager-learning data.
